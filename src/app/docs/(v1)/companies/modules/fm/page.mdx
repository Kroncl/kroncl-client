# 1. «Финансы»

Модуль «Финансы» предоставляет функционал для работы с двумя основообразующими 
подсистемами:

- <Link href='/docs/companies/modules/fm/operations' className={styles.link}>операции</Link>
- <Link href='/docs/companies/modules/fm/debts' className={styles.link}>долговые обязательства</Link>
<br />

# 2. До прочтения модуля

В главах этого модуля используются перекрестные ссылки и термины из следующих разделов документации:

- <Link href='/docs/companies/hrm/employees' className={styles.link}>Сотрудники</Link>: <span className={styles.secondary}>записи сотрудников используются при назначении ответственных за финансовые операции.</span>
<br />
Рекомендуем начать чтение этого модуля после прочтения указанных глав.
